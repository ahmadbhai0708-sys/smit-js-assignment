# SMIT JavaScript Assignments - Complete Coursework

### Overview
This repository contains all JavaScript assignments for the SMIT course. 
All solutions are organized in a single file for easy access.

### Repository Structure
- `assignment1.js` → Contains code for Chapter 1 through Chapter 13
- Each chapter is separated by clear comment blocks for navigation

### How to Run the Code
1.  Open `assignment1.js` 
2.  Uncomment the chapter you want to test
3.  Comment out all other chapters using `//`
4.  Open `index.html` in your browser to view the output
